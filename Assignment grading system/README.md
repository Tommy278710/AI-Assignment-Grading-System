# AI Assignment Grading System

An AI-powered academic evaluation system built with Streamlit and Google's Gemini AI.

## Features

- User Authentication
- Admin Dashboard
- Lecturer Dashboard
- Student Dashboard
- Assignment Submission
- PDF & DOCX Text Extraction
- AI-based Assignment Grading
- Manual Grade Override
- Comments & Feedback
- Analytics Dashboard
- CSV Grade Export

## Technologies

- Python
- Streamlit
- Google Gemini AI
- SQLite
- Plotly
- Pandas
- PyPDF2
- python-docx
- bcrypt

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
